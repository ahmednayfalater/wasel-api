⚡ وصل — Wasel API

نظام إدارة المولدات الكهربائية — Laravel Backend

Base URL — أضفها قبل كل endpoint https://wasel-api-production-0719.up.railway.app/api
📌 الـ requests المحمية تحتاج هذين الـ Headers:
Authorization: Bearer {token} — تحصل على الـ token بعد تسجيل الدخول
Accept: application/json — مطلوب في كل request
دلالة الألوان: عام — بدون login Customer — يحتاج token Provider — يحتاج token + دور provider Admin — يحتاج token + دور admin
🧪 حسابات الاختبار الجاهزة (Seeder)
Admin
Email: admin@wasel.com
Password: admin1234
Provider
Email: provider@wasel.com
Password: provider1234
Customer
Email: customer@wasel.com
Password: customer1234
1. المصادقة — Auth
POST /register/customer تسجيل مواطن عام
Body (JSON)
{
  "first_name": "محمد",
  "last_name": "أبو علي",
  "email": "customer@test.com",
  "password": "password123",
  "password_confirmation": "password123",
  "phone": "0599111111"
}
POST /register/provider تسجيل صاحب مولد عام
Body (form-data) — 3 خطوات في request واحد
--- الخطوة 1: البيانات الشخصية ---
first_name            = خالد
last_name             = النجار
email                 = provider@test.com
password              = password123
password_confirmation = password123
phone                 = 0599222222
company_name          = مولدات النجار
terms_subscr          = يتم الدفع شهرياً  (اختياري)

--- الخطوة 2: بيانات المولد ---
generator_type        = كاتربيلر 2022
generator_powerKW     = 150
generator_gps         = 31.5017,34.4668  (اختياري)
generator_price       = 5.5

--- الخطوة 3: صور الإثباتات ---
proofs[]              = {صورة إثبات الملكية}
proofs[]              = {صورة الهوية}
POST /login تسجيل دخول — يرجع token عام
Body (JSON)
{
  "email": "customer@test.com",
  "password": "password123"
}
POST /forgot-password طلب إعادة تعيين كلمة المرور عام
Body (JSON)
{ "email": "user@test.com" }
POST /reset-password إعادة تعيين كلمة المرور عام
Body (JSON)
{
  "token": "TOKEN_FROM_EMAIL",
  "email": "user@test.com",
  "password": "newpassword123",
  "password_confirmation": "newpassword123"
}
GET /user بيانات المستخدم الحالي مسجّل دخول
لا يحتاج Body
POST /logout تسجيل خروج مسجّل دخول
لا يحتاج Body
2. المولدات — Generators
GET /generators كل المولدات النشطة عام
Query Params اختيارية
?area_id=1&status=active
GET /generators/search بحث عن مولد عام
Query Params
?q=كاتربيلر
GET /generators/compare مقارنة بين مولدين أو ثلاثة عام
Query Params
?ids[]=1&ids[]=2&ids[]=3
GET /generators/{id} تفاصيل مولد عام
لا يحتاج Body
GET /provider/generators مولداتي أنا Provider
لا يحتاج Body
POST /generators إضافة مولد Provider
Body (JSON)
{
  "type": "كاتربيلر 2022",
  "status": "active",
  "gps": "31.5017,34.4668",
  "powerKW": 150,
  "price_KW": 5.5
}
PUT /generators/{id} تعديل مولد Provider
Body (JSON)
{
  "status": "maintenance"
}
DELETE /generators/{id} حذف مولد Provider
لا يحتاج Body
